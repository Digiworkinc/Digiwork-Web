Panduan Demo Presentasi

Gunakan skenario berikut saat mendatangi / presentasi di depan pemilik usaha mebel untuk meyakinkan mereka bahwa sistem Anda sangat tangguh:

  1. Tunjukkan Kemudahan Order: Minta mereka menscan QR code/membuka web ini. Tambahkan mebel, tentukan kustomisasi (warna bantal & finishing kayu), dan buka Keranjang Belanja.
  2. Simulasi Ganti Harga (Real-time): Buka Admin Panel di HP Anda, ubah harga salah satu kursi (misal K-001 dari Rp450k menjadi Rp600k). Tunjukkan perubahan harga terjadi detik itu juga di katalog pembeli tanpa perlu cetak ulang brosur!
  3. Simulasi Database Rusak (Error): Klik tombol Hapus Semua Data di ruang simulasi. Katalog akan langsung kosong. Ini menunjukkan simulasi kegagalan server.
  4. Restore Backup 1-Klik: Yakinkan mereka dengan mengeklik Restore Backup di admin panel. Data katalog kembali normal dan aman seketika dari penyimpanan cloud!
